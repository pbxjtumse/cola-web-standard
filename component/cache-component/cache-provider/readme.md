# 本地缓存实现层
负责 Caffeine 的具体实现。
# 主要能力
| 能力                 | 说明                                   |
| ------------------ | ------------------------------------ |
| 本地缓存 get/put/evict | 操作 Caffeine                          |
| 本地容量控制             | maximumSize / maximumWeight          |
| 本地过期策略             | expireAfterWrite / expireAfterAccess |
| 本地统计               | hitRate / evictionCount              |
| 本地缓存隔离             | 不同 cacheName 使用不同 Caffeine 实例        |
| 本地缓存刷新             | 支持主动刷新或被动刷新                          |
# 适合场景 
| 场景          | 是否适合 |
| ----------- | ---- |
| 字典、枚举、配置类数据 | 很适合  |
| 热点读多写少数据    | 适合   |
| 用户余额        | 不适合  |
| 库存扣减        | 不适合  |
| 状态频繁变化的数据   | 慎用   |
| 跨节点必须强一致的数据 | 不适合  |
# 本地缓存问题
每个应用实例都有一份缓存，天然存在实例间不一致。
